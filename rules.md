# Club Fight Rules Overview

### 1. Setup
*   **Difficulty:** Use one Joker for an easy game or two for a hard game [1, 2].
*   **Prepare Decks:** Separate the **13 clubs** to create a draw pile and shuffle the remaining cards (including Jokers) to create the **grid deck** [1, 2].
*   **Create Grid:** Deal **9 cards** from the grid deck into a face-up **3x3 grid** [1, 2].
*   **Draw Hand:** Draw **4 club cards** from your draw pile into your hand [1, 2].

### 2. Gameplay Loop (The Turn)
A game consists of a maximum of **13 turns**, one for each club card in your deck [3, 4]. Each turn follows these steps:
1.  **Discard and Clear:** Discard a club card (or cards) from your hand to clear cards from the grid using one of **six available actions** [3, 4].
2.  **Check for Blocks:** You cannot use "Blockable Actions" if the grid contains **3 or more cards of a specific suit** [5, 6].
3.  **Refresh the Grid:** Fill empty spaces in the grid from left to right, starting with the top row, using cards from the grid deck [3, 4].
4.  **Refill Your Hand:** Draw club cards until you have **four cards** in your hand again [3, 4].

### 3. Available Actions
*   **Match:** Discard one club to clear all grid cards with the same numerical value [7, 8].
*   **Sum:** Discard one club to clear any number of cards whose values add up exactly to that club's value [7, 8].
*   **Any:** Discard one club to clear any single card; this is one of two actions that can clear a Joker [7, 8].
*   **Line:** Discard one club to remove a full row or column containing a card of that same value [6]. **Blocked if 3+ Diamonds are in the grid** [5, 9].
*   **Up/Down:** Discard a 2–7 of clubs to clear all cards with a *lower* value, or a 7–Q to clear all cards with a *higher* value [10]. **Blocked if 3+ Spades are in the grid** [2, 11].
*   **Run:** Discard one club to remove any number of cards that form a numerical sequence with your discarded card [10]. **Blocked if 3+ Hearts are in the grid** [2, 11].

### 4. Important Rules to Remember
*   **Card Values:** Aces are 1, Jacks are 11, Queens are 12, and Kings are 13 [7, 8].
*   **Jokers:** These have no suit or value and can **only** be cleared using the **Any** or **Line** actions [5-8].
*   **Winning:** You win if you clear the entire grid and there are no more cards remaining in the grid deck [3, 4].
*   **Losing:** You lose if you run out of club cards before the grid and grid deck are empty [3, 4].
