# club-fight
Web adaptation of D. Teuber's Club Fight card game. The game and it's official rules are available here https://boardgamegeek.com/boardgame/410624/club-fight

# Description

Club Fight is a quick solitaire game played with a traditional deck of cards. Using the 13 cards of the club deck, you must clear all the remaining cards of the deck from a 3x3 grid, using six types of actions available to you. However, each of the other suits has the ability to block one of your action types, if cards of the same suit appear at least three times in the grid.
